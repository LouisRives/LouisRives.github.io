Wordle — Multiplayer

◆  JAVA  ·  LOCAL SERVER  ·  MULTIPLAYER  ·  AI BOTS  ◆

✦

This was a group university project with a team of four. The goal was to recreate the popular word-guessing game Wordle as a fully-featured multiplayer experience. Players connect to a local server, meaning anyone on the same network can join and compete against each other in real time. The game also supports AI bot players to fill lobbies when playing solo or with fewer participants.

The game revolves around a point-scoring system: players earn points based on how quickly and accurately they guess the word, encouraging competition across rounds. A lobby system allows players to create or join sessions before the game begins, and a scoreboard tracks everyone's performance throughout the session.

My contribution focused on the AI bot system — designing and implementing the bot players that can join a game when human players are unavailable, complete with word-guessing logic to make them feel like genuine opponents.

Main features:

  • Multiplayer over a local network server
  • Lobby creation and player matchmaking
  • AI bot players for solo or partial lobbies
  • Point-scoring system across rounds
  • Live scoreboard with player rankings
  • Developed in Java
