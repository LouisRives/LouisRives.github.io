Map'Trip

◆  ANDROID  ·  JAVA  ·  GOOGLE MAPS API  ◆

✦

This project was carried out in a team of three during my Erasmus in the Czech Republic. The goal was to create a mobile application for all travellers, regardless of their age. The application works as an interactive map where you can add notes for each country you've visited. The purpose of these notes is to describe your travels — when you went, what you did, and any other memories worth keeping. The application serves as a travel journal so you never forget your adventures.

The application includes three tabs accessible from the bottom navigation bar: the first displays the map, the second is a list of all countries with the ability to edit or delete each note individually, and the last tab is dedicated to options. For my part, I handled the entirety of the first tab, including the integration of the map, retrieving data from the selected country, adding a search bar to find a country, and implementing the note-adding system.

Main features:

  • Mobile application for Android
  • Developed in Java
  • Interactive map using the Google Maps API
  • Persistent note storage — data is saved after the app is closed
