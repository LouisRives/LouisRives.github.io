Map'Trip

This project was carried out in a team of three during my Erasmus in the Czech Republic. The goal was to create a mobile application for all travelers, regardless of their age. The application works as an interactive map where you can add notes for each country you’ve visited. The purpose of these notes is to describe your travels: when you went, what you did, etc. The application serves as a memory book to keep a record of your adventures so that you never forget them.

The application includes three tabs accessible from the bottom navigation bar: the first displays the map, the second is a list of all countries with the ability to edit or delete each note individually, and the last tab is dedicated to options. For my part, I handled the entirety of the first tab, including the integration of the map, retrieving data from the selected country, adding a search bar to find a country, and implementing the note-adding system.

Main features:

  • Mobile application
  • Developed in JAVA
  • Map retrieval using the Google Maps API
  • Note storage system to save them even after the app is closed
