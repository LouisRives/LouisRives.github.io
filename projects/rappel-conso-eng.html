Consumer Recall Application

◆  ANDROID  ·  JAVA  ·  REST API  ◆

✦

This project aims to create a mobile application using the API from the "rappel.conso.gouv.fr" website. The application allows tracking of product recalls, offering several key features: quick and detailed search of recalls, favourite product management, and customisable filters to sort results according to user preferences. It provides a smooth user experience with intuitive navigation and a clear display of essential data, such as associated risks and compensation details.

I mainly worked on creating the database and managing the favourites, including adding and removing items, as well as building the favourites page. I also developed the SecondFragment, enabling the display of recall details and managing their favourite status.

Main features:

  • Mobile application for Android
  • Developed in Java
  • Recall search and display via REST API
  • Detailed recall view
  • Customisable sorting and filters
  • Favourites management with local database
