Consumer Recall Application

This project aims to create a mobile application using the API from the "rappel.conso.gouv.fr" website. The application allows tracking of product recalls, offering several key features: quick and detailed search of recalls, favorite product management, and customizable filters to sort results according to user preferences. It provides a smooth user experience with intuitive navigation options and a clear display of essential data, such as associated risks and compensation details.

I mainly worked on creating the database and managing favorites, including adding/removing items and the favorites page. I also developed the SecondFragment, enabling the display of recall details and managing their favorite status.

Main features:

  • Mobile Application
  • Developed in JAVA
  • Recall search and display
  • Details display
  • Customizable sorting and filters
  • Favorites management
