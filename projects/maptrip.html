Map'Trip

◆  ANDROID  ·  JAVA  ·  API GOOGLE MAPS  ◆

✦

Ce projet a été réalisé en équipe de trois lors de mon Erasmus en République Tchèque. L'objectif était de créer une application mobile destinée à tous les voyageurs, quel que soit leur âge. L'application fonctionne comme une carte interactive sur laquelle vous pouvez ajouter des notes pour chaque pays que vous avez visité. L'intérêt de ces notes est de pouvoir décrire vos voyages — quand vous y êtes allés, ce que vous y avez fait, et tout autre souvenir à conserver. L'application sert de journal de voyage pour ne jamais oublier vos aventures.

L'application comporte trois onglets accessibles depuis la barre de navigation en bas de l'écran : le premier affiche la carte, le deuxième est une liste de tous les pays avec la possibilité d'éditer ou de supprimer chaque note individuellement, et le dernier onglet est dédié aux options. Pour ma part, je me suis occupé de l'intégralité du premier onglet : l'intégration de la carte, la récupération des données du pays sélectionné, l'ajout d'une barre de recherche pour trouver un pays, ainsi que la mise en place du système d'ajout de notes.

Principales caractéristiques :

  • Application mobile Android
  • Développée en Java
  • Carte interactive via l'API Google Maps
  • Système de stockage persistant — les notes sont conservées après fermeture de l'application
